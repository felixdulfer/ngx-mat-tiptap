Example: Using ngx-mat-tiptap-renderer

The ngx-mat-tiptap-renderer component renders TipTap JSON content as HTML. Example usage: 

<ngx-mat-tiptap-renderer
  [content]="form.get('tiptapContent')?.value"
  [cssClass]="'my-content-class'" />

This will display the formatted content from the editor below, using the optional CSS class my-content-class.
